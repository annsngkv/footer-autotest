# footer-autotest
# Тестовое задание

Проект содержит автотест, написанный на Python с использованием Selenium, для проверки футера на сайте https://only.digital.

# Что проверяет автотест:

- Наличие футера на странице  
- Наличие кнопки "Начать проект" внутри футера  
- Наличие ссылки на эл.почту внутри футера  
- Работает на нескольких страницах сайта и учитывает динамические классы футера

# Требования:

- Python 3.x установлен и добавлен в PATH 
- Браузер Google Chrome и ChromeDriver, добавленный в системный PATH
- Git - если используется клонирование через git

# Особенности:
- Автотест учитывает динамические имена классов футера с помощью XPath contains()
- При необходимости можно изменить список страниц для проверки

# Инструкция запуска автотеста на Windows
#1. Клонировать репозиторий через Git
```cmd
cd C:\Users\Имя\Documents
git clone https://github.com/annsngkv/footer-autotest/tree/featurenew-footer
cd footer-autotest
dir
```
#2. Создать виртуальное окружение
```cmd
python -m venv venv
venv\Scripts\activate
```

#3. Установить зависимости
```cmd
pip install -r requirements.txt
```
#4. Запустить автотест
```cmd
python test_footer.py
```
Будет открыта страница Chrome, на которой будет производится поиск футера и его элементов. После завершения теста браузер закроется автоматически.
Результаты появятся в консоли.
