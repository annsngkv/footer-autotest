# footer-autotest
# Тестовое задание

Проект содержит автотест, написанный на Python с использованием Selenium, для проверки футера на сайте https://only.digital.

# Что проверяет автотест:

- Наличие футера на странице  
- Наличие кнопки "Начать проект" внутри футера  
- Наличие ссылки на эл.почту внутри футера  
- Работает на нескольких страницах сайта и учитывает динамические классы футера

# Требования:

- Python 3.7+  
- Библиотека Selenium  
- Браузер Google Chrome и ChromeDriver, добавленный в системный PATH

# Особенности
- Автотест учитывает динамические имена классов футера с помощью XPath contains()
- При необходимости можно изменить список страниц для проверки
